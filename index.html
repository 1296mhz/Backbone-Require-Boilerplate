Backbone  Require  Boilerplate (BRB)

A Rad Backbone.js and Require.js Boilerplate Project.

An opinionated boilerplate project that helps you get started making awesome single page JavaScript web applications.

Video Overview Github »

Created by Greg Franko

Decoupling

Use Require.js to help decouple your JavaScript application into small, self-contained files (modules). No more one huge JavaScript file for you!

Separating Logic

Utilize Backbone.js Collections, Models, and Views to separate your data from the DOM. DOM interaction is expensive (from a performance perspective) so be responsible and separate your logic!

Mobile/Desktop

No one wants to have to write two different applications to provide a usable Mobile Web and Desktop experience. Reuse your logic while also having the flexibility to use a desktop UI framework (Twitter Bootstrap) and a mobile UI framework (jQuery Mobile).

Optimization

Use the Require.js optimizer, r.js, and Almond.js (both written by the amazing James Burke) to minify and concatanate your Mobile Web/Desktop single page application. Let's squeeze out every bit of performance that we can!

Unit Testing

Use the popular BDD JavaScript unit testing framework, Jasmine, to write readable unit tests for your application. Also use other goodies such as the jasmine-jquery plugin to use familiar jQuery assertion selectors.

Node.js

Node.js is used to run the Require.js optimizer on the command line and as a convenience web server for you to get started using the boilerplate. Can anyone say full JavaScript stack?
